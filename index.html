Monolithic Design

High-speed execution within a single address space. Currently implementing context switching for kernel services.

CPIO Initramfs

Compressed early filesystem loaded into RAM. Driver loading and decryption key handling is in progress.

Custom Heap

Implementation of `kmalloc` and `kfree` using a bitmap allocator for efficient physical memory management.
